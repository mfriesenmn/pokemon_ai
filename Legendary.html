Predicting Legendary Status






After running each model 50 times, on average the Random Forest model has the highest average model accuracy between all models. This has not been proving based on statistical significance so that would be something we would look further into if time permitted.



Looking into our last iteration across each model, we broke down are test set into legendary and non-legendary status to understand if a model was more accurate at predicting legendary or failing to predict legendary (false positive). All models were extremely accurate in identifying non-legendary pokemon correctly. KNN had the worst accuracy (66%) of correctly identiying legendary pokemon
